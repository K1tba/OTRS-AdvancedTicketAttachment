# AdvancedTicketAttachment

This is a remastered version of TicketAttachments addon made by Renee Baecker https://github.com/reneeb/opm-TicketAttachments

AdvancedTicketAttachment is a OTRS package, that adds a provides an additional set of functions for working with attachments and also brings a new widget - "AttachmentList" to the system for the AgentTicketZoom overview form.

For technical information please see [this page](doc/en/AdvancedTicketAttachment.pod).
